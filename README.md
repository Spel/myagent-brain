# myagent-brain

Personal knowledge brain for the myagent stack.

This repository is synced to a gbrain Supabase instance and used
by the openclaw/gbrain agent runtime for hybrid RAG search and the
nightly dream cycle.

## Structure

-  — quick captures
-  — long-form notes
-  — auto-generated reports

## Automation

Nightly dream cycle runs via gbrain autopilot (cron inside the cbrain Docker container).
